# Shamo Application

This application is an e-commerce application where users can buy the goods they like. This application is not yet equipped with a payment gateway system and other complex e-commerce systems. The purpose of making this application is as learning material and deepening skills in Flutter as a Front-End Application and Laravel as a Back-End. This application was created by following a course in Indonesia, namely [Build With Angga](https://buildwithangga.com).

## Technology Used

1. Dart - Flutter
2. PHP - [Laravel](https://github.com/Frans-Budi/shamo-api)

## Application Features

1. Authentication - Login
2. Authentication - Register
3. Favorite Products
4. Add to Chart
5. Chats
6. Checkout Products
